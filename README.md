# Tic-Tac-Toe---Python
A simple text-based Tic Tac Toe game with two modes: a 2 Player mode where both players take turns, and a Robot mode where the user plays against a simple computer AI.


FEATURES:
- 3x3 Tic Tac Toe board displayed in the terminal
- Input validation (prevents choosing taken spots or invalid numbers)
- Win and draw detection
- Two game modes (2 players or vs computer)
- Beginner friendly Python code




PUTPOSE OF THIS PROJECT:
This project was created to practice:
- Python basics
- Functions and loops
- Lists and board manipulation
- Simple game logic
- Basic AI decision making
